# C-experiment
c语言程序设计实验课程 

## 代码的规范
- temporary -> temp 暂时的、临时的 -> tmp
- flag 标志 flg
- statistic 统计 -> stat
- count 计数
- increment 加1 -> inc
- decrement 减1 -> dec
- message -> msg
- value -> val
- position -> pos 位置
- addition -> add
- subtractiion -> sub
- multiplication -> mul
- modulus -> mod
- max
- min
